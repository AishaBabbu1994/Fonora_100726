## ✨ Características

- 🗣️ **Texto a Voz** con Web Speech API (Google TTS - ¡Gratuito!)
- 🖱️ **Palabras clickeables** - haz clic en cualquier palabra para leer desde ahí
- 🎯 **Sombreado del segmento** - visualiza el texto que se va a leer
- ⏸️ **Pausas personalizadas** - ajusta el tiempo de pausa entre palabras (0-2s)
- 🌍 **Multilenguaje** - Español, Inglés, Francés, Chino, Portugués, **Árabe**
- 🇸🇦 **Soporte para Árabe** - interfaz RTL y voces árabes
- 🌙 **Modo oscuro/claro** - con persistencia en localStorage
- 📄 **Exportar a PDF** - guarda tus lecturas
- ⚡ **Controles completos** - velocidad, tono, repetición, pausas
- 🎨 **Diseño moderno** - con animaciones y efectos visuales
- 🔒 **100% Gratuito** - sin necesidad de API keys

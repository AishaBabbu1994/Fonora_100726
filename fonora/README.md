**Lector de Texto a Voz Interactivo con IA de Groq**

![TypeScript](https://img.shields.io/badge/TypeScript-5.5.3-blue)
![Tailwind](https://img.shields.io/badge/Tailwind-3.4.6-06B6D4)
![Groq](https://img.shields.io/badge/Groq-API-red)

## ✨ Características

- 🗣️ **Texto a Voz** con IA de Groq
- 🖱️ **Palabras clickeables** - haz clic en cualquier palabra para leer desde ahí
- 🎯 **Sombreado del segmento** - visualiza el texto que se va a leer
- 🌍 **Multilenguaje** - Español, Inglés, Francés, Chino, Portugués
- 🌙 **Modo oscuro/claro** - con persistencia en localStorage
- 📄 **Exportar a PDF** - guarda tus lecturas
- ⚡ **Controles completos** - velocidad, tono, pausa, repetición
- 🔒 **API Key local** - solo se guarda en tu navegador

## 🚀 Inicio Rápido

### 1. Obtener API Key de Groq
1. Ve a [console.groq.com](https://console.groq.com)
2. Regístrate y crea una API Key
3. Copia la clave

### 2. Instalación
```bash
